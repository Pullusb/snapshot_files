# What is snapshot files ?

![image_processing20210205-3-7tj9t](https://github.com/user-attachments/assets/5f355aed-bc6f-48c5-9649-f0c1fbfdf13d)

This simple tool, allows you to always work on the same file, and save a copy of a snapshot eachtime you want, as backups.


# Why is it super-useful ?

It allows you to work always on the same file (which means all your external links targeted to your file won't never break), but if you need to get a previous version, those snapshots can save you !
